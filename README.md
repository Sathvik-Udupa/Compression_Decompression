# Compression_Decompression
A compression-decompression engine that uses Run Length Encoding, Bitmask encoding and direct matching. 


Instructions to run:

- Use original.txt file to store 32 bit data to be compressed.
- Compile SIM.java.
- Run SIM with argument 1 to compress the contents of original.txt file into cout.txt file.
- Run SIM with argument 2 to decompress the contents of cout.txt file into dout.txt file.
